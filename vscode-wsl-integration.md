# VS Code WSL Integration

## What is WSL Integration?

WSL integration allows Visual Studio Code to work directly inside the Ubuntu Linux environment running on Windows.

This helps developers:
- Use Linux commands inside VS Code
- Run tools in a Linux environment
- Work on DevOps projects efficiently

---

## Step 1 — Install VS Code

Download and install VS Code from:
https://code.visualstudio.com/

---

## Step 2 — Install WSL Extension

Open VS Code Extensions and install:

- WSL Extension

Extension Name:
`Remote - WSL`

---

## Step 3 — Open Ubuntu Terminal

Run:

```bash
wsl
```

---

## Step 4 — Open Project in VS Code

Navigate to project folder:

```bash
cd devops-notes
```

Open VS Code:

```bash
code .
```

---

## Benefits of WSL Integration

- Linux-based development environment
- Better DevOps workflow
- Access to Linux tools directly
- Easy Docker and Kubernetes integration
- Seamless terminal experience
