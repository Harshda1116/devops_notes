# Linux Basics

## What is Linux?

Linux is an open-source operating system widely used in servers, cloud computing, DevOps, and software development.

---

## Why Linux is Important in DevOps

- Most servers run Linux
- Lightweight and fast
- Strong command-line support
- Automation friendly
- Used in Docker and Kubernetes

---

## Basic Linux Commands

### Check Current Directory

```bash
pwd
```

### List Files

```bash
ls
```

### Create Directory

```bash
mkdir folder-name
```

### Create File

```bash
touch file-name
```

### Change Directory

```bash
cd folder-name
```

### Remove File

```bash
rm filename
```
