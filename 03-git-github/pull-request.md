# Pull Requests

## What is a Pull Request?

A pull request is used to propose changes to a repository before merging them into the main branch.

---

## Pull Request Workflow

```text
Create Branch
→ Make Changes
→ Commit Changes
→ Push Branch
→ Create Pull Request
→ Review Changes
→ Merge Pull Request
```

---

## Why Pull Requests are Important

- Code review
- Team collaboration
- Better code quality
- Change tracking
- Safer deployments
