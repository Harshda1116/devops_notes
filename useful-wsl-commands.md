# Useful WSL Commands

## Start WSL

```bash
wsl
```

---

## Check Linux Distribution

```bash
wsl -l -v
```

---

## Shutdown WSL

```bash
wsl --shutdown
```

---

## Update Ubuntu Packages

```bash
sudo apt update && sudo apt upgrade -y
```

---

## Check Current Directory

```bash
pwd
```

---

## List Files

```bash
ls
```

Detailed list:

```bash
ls -la
```

---

## Create Folder

```bash
mkdir folder-name
```

---

## Create File

```bash
touch filename.md
```

---

## Remove File

```bash
rm filename.md
```

---

## Remove Folder

```bash
rm -r folder-name
```

---

## Open VS Code in Current Folder

```bash
code .
```

---

## Clear Terminal

```bash
clear
```

---

## Check Ubuntu Version

```bash
lsb_release -a
```
